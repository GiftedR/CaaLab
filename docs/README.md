# CaaLab

**CaaLab** or **C**ross **A**pp Col**lab**orator is a drawing app that aims to allow users of different apps to work on the same project.

**THIS IS A WORK IN PROGRESS**

## Current Designs

The app currently has a plan for 2 modes

### Drawing mode

The drawing mode would be for drawing. Meant to be a mix of design between a few art apps.
